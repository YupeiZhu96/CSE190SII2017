We have to design our digital watermark to fit our expectation. Here are some concepts we need to consider:

Robustness


We have to make sure that the watermark can resist desirable modification so that we can still detect the watermarks even if the original content is changed. If the watermark can only endure only small amount of modification, it is fragile. If the watermark can resist a designated class of transformations, it is robust.

Perceptibility

Original Invisible Visible

It means whether we can detect the watermark perceptually. It is basically how visible watermarks and invisible watermarks are defined.

Capacity


Some watermarks may add extra size to the original data while some may not. For example, the metadata will take extra space and adding logos to pictures will not add extra bits to the original file conceptually.

Embedding method


There are basically 3 kinds of different methods:

1. spread-spectrum: marked signal is obtained by an additive modification. It is not rubost but has a low information capacity.

2. quantization type: marked signal is obtained by quantization. It has low robustness but a high information capacity.

3. amplitude modulation: marked signal is embedded by additive modification which is similar to spread spectrum method, but is particularly embedded in the spatial domain.
